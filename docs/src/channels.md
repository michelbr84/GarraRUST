# Canais

O GarraIA oferece suporte a múltiplos canais de comunicação, permitindo que seus agentes de inteligência artificial interajam com usuários através de diferentes plataformas.

---

## Canais Suportados

### Telegram

Suporte completo com:

* Respostas em streaming (respostas enviadas em tempo real)
* Formatação usando MarkdownV2
* Comandos de bot
* Indicadores de digitação
* Lista de usuários permitidos (allowlist) para controle de acesso

---

### Discord

Suporte completo com:

* Comandos slash (`/comando`)
* Manipulação de mensagens baseada em eventos
* Gerenciamento de sessões por usuário ou canal

---

### Slack

Suporte com:

* Modo Socket (Socket Mode)
* Respostas em streaming
* Sistema de allowlist e pareamento de usuários

---

### WhatsApp

Integração com:

* API Meta Cloud (via webhooks)
* Sistema de allowlist e pareamento
* Comunicação segura entre o GarraIA e o WhatsApp

---

### iMessage

Suporte nativo no macOS com:

* Leitura de mensagens através do banco de dados `chat.db`
* Suporte a conversas em grupo
* Envio de mensagens usando AppleScript
* Integração direta com o sistema iMessage do macOS
